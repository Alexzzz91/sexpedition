## Правила импорта

1. Запрещено импортировать ассеты без явной лицензии.
2. Запрещено импортировать NC/ND лицензии.
3. Для CC-BY/CC-BY-SA обязательно хранить автора, URL, текст attribution.
4. Для каждого импортированного файла добавлять запись в этот реестр.
