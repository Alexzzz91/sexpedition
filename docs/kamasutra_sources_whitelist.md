# Whitelist источников визуалов поз

Цель: источники с бесплатным коммерческим использованием и явной лицензией для веб-версии.

## Разрешенные источники

1. Wikimedia Commons, категория Kama Sutra  
   - URL: https://commons.wikimedia.org/wiki/Category:Kama_Sutra  
   - Лицензия: Public Domain / PD-Art / CC (по конкретному файлу)  
   - Правило: использовать только файлы с явной пометкой PD/CC-BY/CC-BY-SA, исключать NC/ND.

2. Wikimedia Commons (примерные файлы-референсы)
   - https://commons.wikimedia.org/wiki/File:KamaSutra11.jpg
   - https://commons.wikimedia.org/wiki/File:KamaSutra35.jpg
   - https://commons.wikimedia.org/wiki/File:Kamasutra_vatsyayana_Wellcome_L0045293.jpg

3. Internet Archive (многостраничные издания как reference)
   - https://archive.org/details/illustratedkamas0000vats
   - https://archive.org/details/completeillustra0000vats
   - Примечание: для прод-ассетов проверять права на конкретные иллюстрации и статус издания.
