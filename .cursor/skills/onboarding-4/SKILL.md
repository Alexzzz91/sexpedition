# /onboarding-4

## Purpose

Implement copywriting pass focused on value framing and tone.

## Target

- Onboarding UI text
- Early empty states and CTA labels in relevant screens

## Copy goals

1. Explain why use the app:
   - build closeness
   - improve communication
   - keep intimacy intentional.
2. Explain what user gets:
   - shared rituals
   - safer discussion of wishes
   - better planning moments.
3. Keep tone:
   - warm
   - respectful
   - no pressure/judgment.

## Rules

- Avoid clinical or overly explicit language.
- Prefer short, concrete benefits.
- Each screen should have one core message.

## Implementation checklist

- [ ] Onboarding text updated end-to-end
- [ ] CTA labels are clear and human
- [ ] Empty states reinforce value, not guilt
- [ ] Consistent terminology across screens

## Validation

1. Read through full onboarding flow in app.
2. Confirm messaging is consistent and concise.

## Done criteria

- Users can clearly understand how to use app, why it matters, and expected outcomes.
